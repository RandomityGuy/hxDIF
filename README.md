# hxDIF
Never write a dif parser again

# Compilation
## C++
```haxe build-cpp.hxml```
## C#
```haxe build-cs.hxml```
## PHP
```haxe build-php.hxml```
## Python
```haxe build-py.hxml```
## HashLink
```haxe build-hl.hxml```
# Usage
```
Dif.Load(<path>) // Parses the dif at the file location and returns it
Dif.Save(<dif>,<version>,<path>) // Exports the provided dif of target version to the file location
```
